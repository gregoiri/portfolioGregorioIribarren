Mis Proyectos

Carrito de compras

Ver demo Código GitHub

¿De qué se trata?

Aplicación web ASP.NET Core MVC que consiste en una plataforma para una PYME de indumentaria. Permite, por un lado, que el Empleador administre productos, categorías, marcas, empleados y stock en sucursales; y por otro lado, que el Cliente se registre, navegue el catálogo, agregue productos a un carrito, finalice sus pedidos y consulte su historial de compras.

Ingresar como Empleador

¿Qué puedo hacer?

  • ABM de Productos, Categorías, Marcas y Sucursales.
  • Gestionar stock por sucursal.
  • Habilitar / deshabilitar productos.
  • ABM de Empleados.
  • Consultar compras del mes y su detalle.
Ingresar como Cliente (Usuario)

¿Qué puedo hacer?

  • Navegar el catálogo sin iniciar sesión.
  • Agregar productos al carrito y modificar cantidades.
  • Finalizar compra eligiendo sucursal.
  • Acceder al historial de compras.
  • Actualizar datos de contacto.

Tecnologías

  • C#
  • HTML & CSS
  • Entity Framework Core
  • ASP.NET Core MVC
  • SQL Server / SQLite
  • JavaScript

Star Wars App – Frontend

Ver demo

¿De qué se trata?

Aplicación web desarrollada en React y Next.js basada en el universo de Star Wars. Incluye un sistema ABM (Alta, Baja y Modificación) de personajes, junto con funcionalidades interactivas como batallas entre personajes y preguntas tipo trivia.

Acceso y roles

La aplicación permite ingresar con distintos roles: como Administrador, con acceso a la gestión de usuarios y datos del sistema, o como Usuario.
En ambos casos, se pueden utilizar todas las funcionalidades interactivas del juego. Se integra con una API REST propia, incorporando autenticación, manejo de estado y navegación dinámica.

Características principales

  • ABM completo de personajes
  • Gestión de usuarios con rol Administrador
  • Acceso como Usuario o Administrador
  • Batallas interactivas entre personajes
  • Preguntas tipo trivia del universo Star Wars
  • Funcionalidades del juego disponibles para todos los usuarios
  • Consumo e integración con API REST propia
  • Interfaz dinámica y navegación entre vistas

Tecnologías

  • React
  • Next.js
  • JavaScript
  • Context API
  • HTML & CSS
